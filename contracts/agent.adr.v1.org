#+TITLE: Agent ADR Contract v1

* agent-adr-v1
:PROPERTIES:
:CONTRACT_ID: agent.adr.v1
:CONTRACT_KIND: org-elements
:CONTRACT_SCOPE: document
:END:

** What This Produces
This contract validates an architecture decision record. An ADR captures one
decision, its context, consequences, and evidence. Use it as a governing
contract when a plan is mainly about choosing or revising a technical decision.

** adr-has-root
:PROPERTIES:
:ASSERT_ID: adr-has-root
:SEVERITY: error
:END:

#+BEGIN_SRC org-contract
(assert exists
  (headline :summary-contains (title "ADR")))
#+END_SRC

#+BEGIN_SRC jinja2 :name message
ADR files must have an ADR root heading.
#+END_SRC
