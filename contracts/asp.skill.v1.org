* asp-skill-v1
:PROPERTIES:
:CONTRACT_ID: asp.skill.v1
:CONTRACT_ALIAS: ../contracts/asp.skill.v1.org#asp.skill.v1 ../contracts/asp.skill.v1.org
:CONTRACT_BASE: asp.skill.trunk.v1
:CONTRACT_OVERRIDE: subtree
:CONTRACT_KIND: org-elements
:CONTRACT_SCOPE: subtree
:END:

** What This Produces
This contract validates an ASP Org skill subtree that overrides the shared
=asp.skill.trunk.v1= default. The trunk contract defines general skill
quality. This override keeps those requirements and adds ASP Org-specific
requirements for semantic-AST query boundaries, contract/template resources,
and recovery behavior.

The intended output is an Org skill with:
- indexed identity metadata;
- explicit use and non-use boundaries;
- state workflow steps driven by =asp org= query/trace output;
- progressive-disclosure links to contracts and templates;
- validation checks and pressure-scenario guidance;
- adversarial review criteria for model-audited quality;
- provider-boundary rules that prevent SDD, Wendao, or client planning labels
  from becoming hidden query modes.
