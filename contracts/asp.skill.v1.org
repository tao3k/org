** What This Produces
This contract validates an ASP Org skill subtree that overrides the shared
=asp.skill.trunk.v1= default. The trunk contract supplies the package-wide
minimum shape; this contract repeats that baseline and adds ASP Org-specific
checks for contract/template references and recovery behavior.

The intended output is an Org document with:
- a top-level skill heading;
- =SKILL_ID=, =SKILL_DESCRIPTION=, and =CONTRACT_ORG= properties;
- a trigger sentence that starts with =Use this skill=;
- explicit contract/template references using sibling-relative package paths;
- query or workflow commands preserved in source blocks;
- rules that make the provider boundary explicit;
- recovery or validation instructions for later agents.

** skill-has-root-heading
:PROPERTIES:
:ASSERT_ID: skill-has-root-heading
:SEVERITY: error
:END:
#+BEGIN_SRC org-contract
assert exists headline where at($scope)
#+END_SRC
#+BEGIN_SRC jinja2 :name message
ASP skill files must have a top-level skill heading.
#+END_SRC
