** What This Produces
This contract validates an ASP Org skill file. It is intentionally a shared
skill-package contract: concrete skills describe their domain, while this
contract checks the common authoring shape that makes a skill usable by agents.

The intended output is an Org document with:
- a top-level skill heading;
- a =CONTRACT_ORG= reference to this contract;
- a trigger sentence that starts with =Use this skill=;
- explicit contract/template references using sibling-relative package paths;
- query or workflow commands preserved in source blocks;
- rules that make the provider boundary explicit;
- recovery or validation instructions for later agents.
