#+TITLE: Agent BDR Contract v1

* agent-bdr-v1
:PROPERTIES:
:CONTRACT_ID: agent.bdr.v1
:CONTRACT_KIND: org-elements
:CONTRACT_SCOPE: document
:END:

** What This Produces
This contract validates a Business Development Representative workflow
specification. A BDR record captures target segment context, prospect/account
criteria, outreach hypotheses, qualification signals, and handoff evidence. Use
it as a governing contract when a plan is mainly about business-development
representative work rather than software behavior.

** bdr-has-root
:PROPERTIES:
:ASSERT_ID: bdr-has-root
:SEVERITY: error
:END:

#+BEGIN_SRC org-contract
(assert exists
  (headline :summary-contains (title "BDR")))
#+END_SRC

#+BEGIN_SRC jinja2 :name message
BDR files must have a BDR root heading.
#+END_SRC
