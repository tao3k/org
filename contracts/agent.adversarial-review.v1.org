* agent-adversarial-review-v1
:PROPERTIES:
:CONTRACT_ID: agent.adversarial-review.v1
:CONTRACT_KIND: org-elements
:CONTRACT_SCOPE: document
:END:

** What This Produces
This contract validates a model-authored adversarial review receipt. The
machine contract does not decide whether prose quality is good; it requires the
reviewer to record a bounded challenge, explicit criteria, concrete findings,
evidence, and a verdict that later agents can inspect.

Use this when structural contracts pass but content quality still needs a
reflection pass. The target may be a skill, template, plan, or other durable Org
artifact.

:PROPERTIES:
:ASSERT_ID: adversarial-review-has-root
:SEVERITY: error
:END:

  (headline :at $scope))
#+END_SRC

#+END_SRC
