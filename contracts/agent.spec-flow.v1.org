:END:

** What This Produces
This contract validates a specification authoring flow. The flow is used when a
new theory or governing specification family must be introduced under
=languages/org=.

The flow keeps creation order explicit:
- define the theory and boundary;
- create the contract;
- create the template;
- create at least one example;
- register the contract and template in the skill and template index;
- run contract trace, lint, and adversarial review.
