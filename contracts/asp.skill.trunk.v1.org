:CONTRACT_SCOPE: subtree
:END:
** What This Produces
This contract validates the shared trunk shape for ASP Org skill subtrees. It
maps mature Markdown skill conventions such as name, description, trigger,
workflow, rules, and examples into Org properties, headings, links, and source
blocks.

Set this contract as a package or file default with =#+PROPERTY: CONTRACT_ORG=.
Concrete skill contracts may override this trunk from a skill subtree by setting
their own =CONTRACT_ORG= property. Override contracts must repeat any trunk
requirements they still want enforced.

** trunk-has-root-heading
:PROPERTIES:
:ASSERT_ID: trunk-has-root-heading
:SEVERITY: error
:END:
#+BEGIN_SRC org-contract
assert exists headline where at($scope)
#+END_SRC
#+BEGIN_SRC jinja2 :name message
ASP skill subtrees must have a root heading.
#+END_SRC
