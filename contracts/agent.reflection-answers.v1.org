* agent-reflection-answers-v1
:PROPERTIES:
:CONTRACT_ID: agent.reflection-answers.v1
:CONTRACT_KIND: org-elements
:CONTRACT_SCOPE: subtree
:END:

** What This Produces
This contract validates a reflection question table whose answers are not
empty. The structural contract owns the table/value flow: the model still writes
the answer, but the provider must expose table cells so the contract can reject
missing values.

The expected shape is a =Reflection Questions= or =Closure Questions= subtree
with a two-column Org table:
- =Question=
- =Value=

At least one non-header =Value= cell must be non-empty.

** reflection-has-question-table
:PROPERTIES:
:ASSERT_ID: reflection-has-question-table
:SEVERITY: error
:END:
#+BEGIN_SRC org-contract
assert exists table where descendant_of($scope)
#+END_SRC
#+BEGIN_SRC jinja2 :name message
Reflection answer receipts must include an Org table.
#+END_SRC

:SEVERITY: error
:END:
