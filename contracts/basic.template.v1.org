* basic-template-v1
:PROPERTIES:
:CONTRACT_ID: basic.template.v1
:TITLE: Basic Org Template Metadata v1
:CONTRACT_KIND: org-elements
:CONTRACT_SCOPE: document
:END:

** What This Produces
This contract validates reusable document-level metadata that can be combined
with purpose-specific contracts through multiple =CONTRACT_ORG= declarations.
It is intentionally generic: task, plan, SDD, BDD, TDD, BDR, and PRD semantics
belong to their own contracts.

The paired template provides:
- a document title keyword;
- a document date keyword whose template value is precise to seconds;
- a document author keyword;
- document-level =#+FILETAGS= metadata for scope and registry grouping;
- document-level =#+PROPERTY= metadata for contract identity and provenance.

** basic-template-has-title
:PROPERTIES:
:ASSERT_ID: basic-template.has-title
:SEVERITY: error
:END:
#+BEGIN_SRC org-contract
(assert exists
  (and
    (keyword :context "metadata")
    (= (summary key) "TITLE")))
#+END_SRC
#+BEGIN_SRC jinja2 :name message
Basic templates must include a document-level #+TITLE keyword.
#+END_SRC
