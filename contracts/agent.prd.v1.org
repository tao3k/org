* agent-prd-v1
:PROPERTIES:
:CONTRACT_ID: agent.prd.v1
:CONTRACT_KIND: org-elements
:CONTRACT_SCOPE: document
:END:

** What This Produces
This contract validates a product requirements document. A PRD records the
problem, users, requirements, non-goals, and success metrics. Use it as a
governing contract when a plan is mainly about product intent or user value.

** prd-has-root
:PROPERTIES:
:ASSERT_ID: prd-has-root
:SEVERITY: error
:END:

#+BEGIN_SRC org-contract
(assert exists
  (headline :summary-contains (title "PRD")))
#+END_SRC

#+BEGIN_SRC jinja2 :name message
PRD files must have a PRD root heading.
#+END_SRC
