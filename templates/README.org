* Templates
This directory contains language-facing Org authoring templates. Provider
crates such as =languages/orgize= implement parsing and evaluation; reusable
agent templates and their public contracts live here.

Use this template when an agent needs to leave a recoverable plan ledger after
work is done. The paired contract verifies identity, objective, scope, status,
recovery, lifecycle, validation, and evidence sections.
